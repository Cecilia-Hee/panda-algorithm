# ☆动态规划解题套路

## 套路
1. 明确状态，一种状态的就是一维的dp-table，两种状态的就是二维的dp-table，不过二维的dp-table可以简化为一维的；
2. 明确选择，要么选，要么不选
3. dp-table的状态方程如何确定，才是最难的；
4. dp-table的解法，也可以用递归+备忘录解决
